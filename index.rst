
Anyswap DEX User Guide 
======================

Anyswap is a fully decentralized cross chain swap protocol, based on Fusion DCRM technology, with automated pricing and liquidity system. Anyswap is a decentralized application running on the `Fusion`_ , `Binance Smart Chain`_ , `Ethereum`_ and `Fantom`_ blockchains. The first application from Anyswap is a DEX (Decentralized Exchange), which is called `anyswap.exchange`_. This is the User Guide for the DEX.

A good introductory article about Anyswap can be found here `A Comprehensive Review of the Cross Chain DEX Anyswap`_ in U.Today 7th Sept. 2020
