
Anyswap DEX User Guide 
======================

Anyswap is a fully decentralized cross chain swap protocol, based on Fusion DCRM technology, with automated pricing and liquidity system. Anyswap is a decentralized application running on the `Fusion`_ blockchain. The first application from Anyswap is a DEX (Decentralized Exchange), which is called `anyswap.exchange`_ This is the User Guide for the DEX.
